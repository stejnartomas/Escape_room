# 🎮 Escape Room
Unreal Engine 5 projekt - 3D hororová escape room hra
---
## 📖 O hře
Součástí hry je:
- AI postava s detekcí hráče
- UI menu a herní widgety
- zvukové efekty a atmosféra herního prostředí

## 🕹️ Ovládání
- "WASD" - pohyb hráče
- "Myš" - pohyb kamery 
- "Levé tlačítko na myši" - interakce
- "I" - inventář
- "ESC" - vypnutí hry
(Ovládání lze upravit ve hře přímo v menu -> Controls)
